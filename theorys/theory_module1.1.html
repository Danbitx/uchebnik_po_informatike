Системы счисления

Определение

Система счисления — это символический метод записи чисел, позволяющий представить множество чисел с помощью письменных знаков и обеспечивающий уникальное или стандартное представление каждого числа.

Виды систем счисления

1. Позиционные системы

Значение цифры зависит от её положения в числе. Основные примеры:

  • Двоичная (2) Алфавит: 0, 1 
    1010112 = 1×25 + 0×24 + 1×23 + 0×22 + 1×21 + 1×20 = 4310
  • Восьмеричная (8) Алфавит: 0-7 
    15738 = 1×83 + 5×82 + 7×81 + 3×80 = 89110
  • Десятичная (10) Алфавит: 0-9 
     Представить число 5310 в двоичную систему
                    53 ÷ 2 = 26 (остаток 1)
                    26 ÷ 2 = 13 (остаток 0)
                    13 ÷ 2 = 6  (остаток 1)
                    6 ÷ 2 = 3   (остаток 0)
                    3 ÷ 2 = 1   (остаток 1)
                    1 ÷ 2 = 0   (остаток 1)
                    
        Результат: 1101012
  • Шестнадцатеричная (16) Алфавит: 0-9, A-F 
    7B16 = 7×161 + 11×160 = 12310

2. Непозиционные системы

Значение цифры не зависит от позиции. Пример:

Римская система: XL = 40 (L = 50, X = 10)

Специальные системы

  • Факториальная - основана на факториалах
  • Фибоначчи - запрещены две единицы подряд
  • Штерна-Броко - для рациональных чисел
  • Классов остатков - китайская теорема об остатках

Смешанные системы

Основание меняется для разных разрядов. Пример:

Время: 12:34:56 (часы:основание 24, минуты:60, секунды:60)
